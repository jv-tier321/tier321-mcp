## Frontmatter

The frontmatter parser in `blog-loader.ts` is intentionally minimal. It supports:

- Single-line string values (quoted or unquoted)
- Boolean `true` / `false`
- Array values inside `[...]` with single-line members

Multi-line values, YAML flow sequences, and nested objects are not supported. That's by design — the parser is a few dozen lines and has no runtime dependencies.
